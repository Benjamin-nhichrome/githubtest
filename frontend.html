HTML Element voor Links

Ik heb dat links worden gemaakt met behulp van het -element. Het -element gebruikt het href-attribuut om de pagina aan te geven waarnaar u linkt.

HTML Element voor afbeeldingen

Ik heb element wordt gebruikt om afbeeldingen aan een webpagina toe te voegen. U moet altijd een src-attribuut opgeven om de bron van een afbeelding aan te geven en een alt-attribuut om de inhoud van een afbeelding te beschrijven.

HTML Element voor tabellen

Het table element wordt gebruikt om tabellen aan een webpagina toe te voegen. Rij voor rij wordt een tabel uitgetekend. Elke rij wordt gemaakt met het element.

HTML Element voor formulieren

Wanneer u informatie van bezoekers wilt verzamelen, heeft u een formulier nodig, dat zich in een

- element bevindt. Informatie uit een formulier wordt verzonden in naam/waardeparen
